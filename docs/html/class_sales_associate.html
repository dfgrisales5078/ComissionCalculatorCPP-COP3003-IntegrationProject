void SalesAssociate::AccessoryCommissions ( )
inlineoverridevirtualnoexcept

Overwritten function to calculate sales associate's accessory commissions.

The total accessory commission is the dollar amount sold times the percentage of commission paid depending on goal reached. If sales associate reached goal 1, commission is 16% of the dollar amount sold. If sales associate reached goal 2, commission is 14% of the dollar amount sold.

Implements Employee.

◆ NewAccountCommissions()

void SalesAssociate::NewAccountCommissions ( )
inlineoverridevirtualnoexcept

Overwritten function to set sales associate's new account commissions.

The total new account commission is the number of new accounts sold times the commission per account depending on goal reached. If sales associate reached goal 1, commission is $9 per new account. If sales associate reached goal 2, commission is $7 per new account.

Implements Employee.
